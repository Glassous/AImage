# AImage

AImage 是一款功能强大的 Android AI 图像生成应用，使用 Jetpack Compose 构建。它允许用户连接到多个主流 AI 图像模型提供商，在一个统一的界面中管理和生成图像。

## ✨ 核心功能

* **多供应商支持**：支持配置和切换多个 AI 图像 API，包括 Google Gemini、阿里通义千问、豆包、MiniMax 和 OpenRouter。
* **灵活的模型配置**：用户可以为每个供应商添加、编辑和管理多个模型条目，并单独设置 API Key。
* **多窗口界面**：主界面支持创建多个聊天窗口，允许用户同时与不同模型交互或保留多个会话。
* **提示词助写**：
    * 提供一个专门的“提示词助写”工具，帮助用户通过预设参数（如光圈、焦段、光线、风格等）构建复杂的提示词。
    * 支持使用 AI（需单独配置）对提示词进行润色或翻译。
* **云端同步**：
    * 支持通过配置阿里云 OSS (Object Storage Service) 实现跨设备自动同步。
    * 自动同步模型配置、API Key、历史记录元数据和提示词助写设置。
    * 支持在应用启动时、配置变更时以及增删历史记录时自动触发后台同步。
* **本地备份与恢复**：
    * 提供完整的本地备份功能，可将所有模型配置、API Key 和全部历史记录（包含图片）导出为单个 JSON 文件。
    * 支持从备份 JSON 文件中恢复所有数据。
* **历史记录管理**：
    * 自动保存每一次生成的历史（包括提示词、模型、图片和时间戳）。
    * 侧边栏提供历史记录浏览和删除功能。
* **现代化 UI**：
    * 完全使用 Jetpack Compose 和 Material 3 构建，支持动态色彩。
    * 支持浅色、深色和跟随系统的主题模式。
    * 内置全屏图片预览器，支持手势缩放、平移和旋转。

## 🤖 支持的 AI 供应商

您可以在“设置” > “模型配置”中为以下供应商配置 API Key 和模型：

* **Google** (Gemini)
* **豆包** (Doubao / Volcengine)
* **通义千问** (Qwen / Alibaba)
* **MiniMax**
* **OpenRouter**

## 🛠️ 技术栈

* **语言**：100% Kotlin
* **UI**：Jetpack Compose, Material 3
* **网络**：Retrofit, OkHttp
* **异步**：Kotlin Coroutines & Flow
* **图片加载**：Coil
* **云存储**：Alibaba Cloud OSS Android SDK

## 🚀 如何使用

1.  **配置模型**：
    * 进入“设置” > “模型配置”。
    * 选择一个供应商（例如 Qwen）。
    * 填入您的 API Key。
    * 点击“新增”来添加一个模型，填入服务商提供的模型名称（例如 `qwen-image-plus`）。
2.  **设置默认模型**（可选）：
    * 在“设置”中，点击“默认模型”，选择一个您刚配置好的模型。
3.  **开始生成**：
    * 返回主屏幕，应用会自动使用您的默认模型创建一个窗口（如果设置了默认模型）。
    * 或者，点击右上角的 `+` 按钮，从可用模型列表中选择一个来创建新窗口。
    * 在底部的输入框中输入提示词，点击发送按钮。
4.  **同步数据**（可选）：
    * 进入“设置” > “云端同步（OSS）” > “配置OSS”。
    * 填入您的 OSS 配置信息并保存。
    * 您可以开启“自动同步”开关，或手动执行“全面上传”或“全面下载”。
